Schema

Schema 是数据库集合的模型骨架，定义了集合中的字段的名称和类型以及默认值等信息。

Schema.Type

  • Node 中的基本数据类型都属于 Schema.Type
    • String (字符串)
    • Date (日期)
    • Number(数值)
    • Boolean(布尔)
    • Buffer(二进制)
    • null
    • Array(数组)
  • Mongoose 定义的自己的类型
    • ObjectId(主键)
    • arrOfString([String])
    • arrOfNumber([Number])
    • arrOfDate([Date])
    • arrOfBugger([Buffer])
    • arrOfBoolean([Boolean])
    • arrOfObjectId([Schema.types.ObjectId])
    • nested({name: String}) 内嵌文档
